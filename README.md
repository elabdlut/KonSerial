# KonSerial - 现代化轻量化串口调试工具

一款基于 Tauri + Vue3 和 Rust + TypeScript 构建的现代化、轻量化的串口调试工具。该项目结合了VOFA+和串口调试助手的功能，提供了一个集数据收发、波形显示、协议支持及自动化脚本于一体的综合调试平台。

## 功能特性

### 核心功能
- **多串口同时管理**：支持同时打开多个串口，通过标签页切换不同串口连接
- **数据收发**：支持串口数据的发送和接收，支持十六进制发送和显示
- **波形显示**：实时显示串口发来的数据波形图（支持多种通信协议）
- **文件发送**：支持以文件形式发送串口数据
- **自定义脚本**：支持构建自定义串口发送脚本进行自动化发送
- **数据保存**：支持将接收到的串口数据保存到文件中
- **多协议调试**：支持 TCP/UDP 调试和蓝牙串口调试

### 用户体验
- **现代化UI**：使用 Vue 3 构建的现代化、响应式用户界面
- **标签页式设计**：灵活的标签页界面，可同时管理多个串口连接
- **配置记忆**：自动记忆上次使用的串口参数，重启后无需重新配置
- **跨平台**：基于 Tauri 框架，支持 Windows、macOS 和 Linux 平台

## 技术栈

- **前端**：Vue 3 + TypeScript
- **后端**：Rust
- **框架**：Tauri
- **构建工具**：Vite + pnpm
- **串口库**：serialport (Rust)

## 安装与运行

### 环境要求
- Node.js (推荐最新LTS版本)
- Rust (最新稳定版)
- pnpm

### 开发环境搭建
```bash
# 克隆项目
git clone https://github.com/sratle/KonSerial.git
cd KonSerial

# 安装依赖
pnpm install

# 启动开发服务器
pnpm tauri dev
```

### 生产构建
```bash
# 构建为平台对应的应用程序
pnpm tauri build

# 构建后可在 src-tauri/target/release/bundle/ 目录下找到安装包
```

## 使用说明

### 多串口管理
1. 点击“+”按钮或快捷键创建新的串口标签页
2. 每个标签页可独立配置和连接不同的串口
3. 支持同时打开多个串口进行数据对比和调试
4. 通过标签页切换快速访问不同的串口连接

### 串口配置
1. 连接设备到电脑
2. 在设备管理器中确认串口号
3. 在应用中选择对应串口号和波特率

### 数据收发
- 在发送区域输入数据，点击发送按钮
- 支持ASCII和十六进制格式发送
- 接收区域会实时显示收到的数据

### 波形显示
- 应用支持多种数据协议的解析和波形显示
- 实时图表显示数据变化趋势
- 支持多通道数据同时显示

### 文件发送
- 选择本地文件，支持多种格式
- 配置发送参数后开始发送文件内容

### 脚本功能
- 使用内置脚本编辑器创建自定义发送逻辑
- 支持定时发送、条件判断等高级功能

### 数据保存
- 可将接收的数据保存为文本文件
- 支持多种格式导出

## 协议支持

- 串口协议 (UART/RS232/RS485)
- TCP 协议
- UDP 协议
- 蓝牙串口协议
